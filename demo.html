What changes across the 3 stages?

  • Prompt User: asks AI for an answer directly.
  • Prompt Engineer: improves wording with audience, context, and structure.
  • System Architect: designs how the AI should behave, reason, format, and check quality.

Why this is useful

The project teaches that AI quality depends not only on what you ask, but on how you define the surrounding system: task framing, constraints, role design, output expectations, and workflow logic.
